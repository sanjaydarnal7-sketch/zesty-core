# ZestyBrain File Naming Rules

## Purpose

This document defines how every memory file inside ZestyBrain must be named.

The goal is consistency, fast searching, AI compatibility, and clean organization.

---

# Global Rules

- One topic = One file
- Use English only for filenames
- Use "_" instead of spaces
- Never use special characters
- Keep filenames short
- Never overwrite an existing file without confirmation
- Never create duplicate files

# AI Memory Rules

When saving new information:

1. Search existing files first.
2. If the topic already exists, update it.
3. If it does not exist, create a new file.
4. Ask before deleting files.
5. Never edit system files.
6. Always keep the vault organized.
