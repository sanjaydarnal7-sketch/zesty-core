## Rules

1. Never modify main.py automatically.
2. Never modify index.html automatically.
3. All permanent memory is stored inside this vault.
4. Every topic must be saved as a separate Markdown (.md) file.
5. One file = One topic.
6. Never overwrite existing files without confirmation.
7. Ask before deleting any file.
8. Always search the vault before answering.
9. If information is not found locally, use internet knowledge (when available) and clearly distinguish it from local knowledge.
10. Keep folder structure clean and organized.
